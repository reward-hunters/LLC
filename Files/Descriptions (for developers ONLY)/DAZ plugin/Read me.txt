1. Скачать с сайта DAZ последнюю версию и установить её;
2. Распаковать архив (много весит, кинул на гугл в папку работа). Прочитать pdf, там же находится лицензионный ключ. Установить. Будет установлен плагин + старая версия HeadShop (10.2). Для проверки - просто заменить в папке DAZ abalone на новую версию.
3. В папке DAZ/plugins - заменить dll файл.
4. Запустить DAZ, кинуть на сцену Generic модель. Edit - abalone plugin.
